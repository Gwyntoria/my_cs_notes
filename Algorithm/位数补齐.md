# 位数补齐

## 向上补齐

```c
// 按4补齐
if (stride % 4) {
    stride = (stride & 0xfffc) + 4;
}
```

`0xc`的二进制表示为`0b1100`，如果`stride`非 4 的倍数，经过按位与运算，最后两位会变为 0，即`stride`的值减去了 1 或 2 或 3，按位与运算的结果是，4 的倍数里小于 stride 且最接近 stride 的值。

同理可做按 2 补齐和按 8 补齐等

```c
// 按2补齐
int d = 19;
if (d % 2) {
    d = (d & 0xfffe) + 2;
}

// 按8补齐
int e = 20;
if (e % 8) {
    e = (e & 0xfff8) + 8;
}
```

运用**按位与**运算来进行补齐的方法，只适用于 2 的指数幂的补齐
