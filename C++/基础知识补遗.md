# 基础知识补遗

## union 和 struct 的大小问题

UNION 变量所占用的内存大小 = 其中**最长成员**的内存大小。

```c
union package {
    char head;
    int body;
};
```

联合体 package 的最大内存大小是 int 的内存大小，即 4 字节。

对于结构体：

```c
struct message {
    char id;
    int crc;
    union package pack;
};
```

c 一般默认是 4 字节对齐的，不足 4 字节的要补齐到 4 字节。

（以 O 表示成员本身所占的内存，以 X 表示补齐的字节。）

| char | int  | union package |
| ---- | ---- | ------------- |
| OXXX | OOOO | OOOO          |

一共 12 字节。
